# 资产生命周期

## 发现

1. 加载 (Load)：锁定探测空间。
2. 剪枝 (Prune)：执行物理边界排除。
3. 采样 (Sample)：捕获实体特征快照。
4. 映射 (Map)：应用分类逻辑（Classification）。
5. 发射 (Emit)：交付标准化资产实体清单。

## 下线

1. 弃用（Deprecate）：标记资产为“不推荐使用”，但暂时保留记录，通知下游调用方进行迁移。
2. 归档（Archive）：将资产元数据移入冷存储（历史库），解除其在活跃搜索中的可见性。
3. 清理（Purge）：彻底删除资产库中的逻辑条目。对于高敏感资产，还需确保对应的物理存储已被安全擦除。
4. 切断（Sever）：断开该资产与其他业务实体、权限策略或关联链路的所有耦合。
