# 数字资产治理

数字资产治理是围绕数字资产契约建立的一整套管理机制，涵盖资产的发现、注册、验证等生命周期管理过程。

## 治理架构

```
┌─────────────────────────────────────────────────────────────┐
│                    数字资产契约 (Contract)                   │
│                   「是什么」—— 资产的事实源                    │
└─────────────────────────────────────────────────────────────┘
                              │
        ┌─────────────────────┼─────────────────────┐
        │                     │                     │
        v                     v                     v
┌───────────────┐    ┌───────────────┐    ┌───────────────┐
│   发现过程     │    │   运行时状态   │    │   验证过程     │
│  Discovery    │    │   Registry    │    │  Validation   │
└───────────────┘    └───────────────┘    └───────────────┘
```

## 治理内容

- [数字资产契约](contract.md) - 定义资产的组成和结构
- [数字资产发现](discovery.md) - 自动识别和提取资产
- [数字资产运行时状态管理](registry.md) - 维护资产的运行时状态
- [数字资产验证](validation.md) - 确认资产的完整性和一致性
